"# Dodge_ai_task" 



This project is a full-stack application that visualizes your database in an interactive **graph format (nodes and edges)** and includes an integrated **AI assistant** to help with data analysis.

---

## 🚀 Features

- 🔗 **Graph Visualization**
  - Displays database structure as nodes and edges
  - Interactive UI for exploring relationships between data

- 🤖 **AI Assistant Panel**
  - Located on the right side of the interface
  - Ask questions about your data
  - Get instant insights and analysis

- ⚡ **Full-Stack Architecture**
  - Backend powered by Python
  - Frontend built with modern JavaScript tools

---

## 🛠️ Tech Stack

- **Frontend:** react
- **Backend:** Python
- **Visualization:** Graph-based UI (nodes & edges) using python  
- **AI Integration:** Built-in assistant for data queries using gemini

---

## 📦 Setup Instructions

### 🔹 Backend Setup

1. Navigate to the backend directory:
   ```bash
   cd backend
   ```
Create a virtual environment:

python -m venv venv
Activate the virtual environment:

```
venv\Scripts\activate

```
Install dependencies:

```
pip install -r requirements.txt
```
Run the backend server:
```
python main.py
```
🔹 Frontend Setup

Navigate to the frontend directory:
```
cd frontend
```
Install dependencies:
```
npm install
```
Start the development server:
```
npm run dev
```

▶️ Usage
- Open your browser after starting the frontend.
- View your database in graph format.
- Interact with nodes and edges to explore relationships.
- Use the AI assistant panel on the right to:
- Ask questions
- Perform data analysis
- Get insights instantly
#📌 Notes

- Make sure both backend and frontend servers are running simultaneously.
- Ensure all dependencies are properly installed before running the project.
